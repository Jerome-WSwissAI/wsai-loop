# wsai-loop

Single Cursor plugin. One command: `/wsai-loop`.

```bash
git clone https://github.com/Jerome-WSwissAI/wsai-loop.git ~/.cursor/plugins/local/wsai-loop
```

Then **Developer: Reload Window**.

- Sources must be non-AI URLs + excerpts
- Research generates todolist; every `TD*` must validate
- Runtime: `WSAI_LOOP_ROOT` or `./.wsai-loop`

Docs: https://cursor.com/docs/plugins · https://cursor.com/docs/hooks
