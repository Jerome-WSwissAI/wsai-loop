# wsai-loop

Cursor (+ Claude) plugin: goal → non-AI research → todos → lexeme gate → one-shot.

```bash
git clone https://github.com/Jerome-WSwissAI/wsai-loop.git ~/.cursor/plugins/local/wsai-loop
```

Reload Window. One slash: `/wsai-loop`.

- Sources: non-AI URL + excerpt
- Todos from research; each `TD*` validated
- `ctrl-lexeme`: no character without a reason
- Runtime: `WSAI_LOOP_ROOT` or `./.wsai-loop`

https://cursor.com/docs/plugins · https://cursor.com/docs/hooks · https://code.claude.com/docs/en/plugins
